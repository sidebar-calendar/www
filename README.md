# Sidebar Calendar Website

이 폴더는 Sidebar Calendar 익스텐션의 공식 랜딩 페이지와 개인정보처리방침을 포함하고 있습니다.

## 구조
- `index.html`: 서비스 소개 및 다운로드 링크가 있는 메인 랜딩 페이지
- `privacy/index.html`: 한/영 다국어를 지원하는 개인정보처리방침 페이지

## 배포 방법 (GitHub Pages)
이 웹사이트는 GitHub Pages를 통해 배포됩니다.
1. 리포지토리의 **Settings > Pages** 메뉴로 이동합니다.
2. **Build and deployment > Source**를 `Deploy from a branch`로 설정합니다.
3. **Branch**를 `main` (또는 `master`)으로 선택하고, 폴더를 **`/docs`**로 지정한 뒤 **Save**를 누릅니다.

## 수정 방법
- 랜딩 페이지의 내용을 수정하려면 `index.html`을 편집하세요.
- 개인정보처리방침 내용을 수정하려면 `privacy/index.html`을 편집하세요.
